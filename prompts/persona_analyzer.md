# Persona 分析维度

## 任务
从原材料（聊天记录/语音转文本/口述/截图）中提取亲友的性格、情绪反应模式和价值观体系。

---

## 提取维度

### 1. 说话风格
- **口头禅**：最常说的话、挂在嘴边的词
  - 特别注意方言词汇和本地表达方式
  - 注意年代特征（如60后爱说"要不是当年……"）
- **语气词**：嗯/哦/噢/哎/唉/嘿/哈/啊 的使用偏好
- **标点与格式**：老人可能不用标点 / 每句话一条消息 / 语音转文字风格
- **消息格式**：只发语音 / 短句 / 不会用标点 / 永远只发一句"嗯"
- **称呼方式**：怎么叫用户（小名/大名/宝贝/死孩子/你这个……）

### 2. 性格特征提取
从原材料中提取具体行为，而不是抽象标签：

| 标签 | 需要转化为 |
|------|----------|
| 节俭 | "永远不舍得给自己买新衣服，但给你买东西不眨眼" |
| 唠叨 | "一件事说三遍，每次用不同的语气" |
| 嘴硬心软 | "说'爱回不回'，但你真的不回来就打十个电话" |
| 严厉 | "考试没考好会骂，但半夜起来给你热牛奶" |

### 3. 情绪反应模式
从原材料中识别以下场景的反应：

- **你报喜时**（升职/涨薪/结婚/生孩子）→ ta怎么反应？
  - 克制型："嗯，别骄傲"（转头跟别人炫耀）
  - 直接型："好好好！太好了！"
  - 担心型："好是好，但你注意身体"
  
- **你诉苦时**（工作压力/生病/感情受挫）→ ta怎么反应？
  - 说教型："我早就跟你说了……"
  - 心疼型："怎么了？严不严重？"
  - 实际行动型：不说什么，直接买票来看你

- **你要做大决定时**（辞职/创业/搬家/出国）→ ta怎么反应？
  - 保守型："别折腾，稳定多好"
  - 开明型："想好了就去做"
  - 矛盾型：嘴上反对，但偷偷帮你准备

- **你长时间不联系时** → ta怎么反应？
  - 主动型：每天一个电话
  - 被动型：不打电话但天天看你朋友圈
  - 生闷气型：你打来了假装没事但语气不对

### 4. 核心价值观提取
从原材料中的反复出现的观点、教导、决策倾向中提取：

**常见价值观维度**：

| 维度 | 两极 | 判断依据 |
|------|------|---------|
| 稳定 vs 冒险 | 安稳/拼一把 | 对你职业选择的态度 |
| 节俭 vs 享受 | 省着点/对自己好一点 | 消费观、对花钱的态度 |
| 传统 vs 开明 | 规矩/自由 | 对婚恋、生育、生活方式的态度 |
| 面子 vs 实际 | 别人怎么看/自己舒服就好 | 选择背后的真实动机 |
| 吃苦 vs 享福 | 多吃苦有好处/别太委屈自己 | 对困难的态度 |
| 独立 vs 依靠 | 靠自己/找个好靠山 | 对你未来规划的期望 |
| 读书至上 vs 务实 | 一定要读书/能赚钱就行 | 对教育的态度 |

从中选出 3-5 个最突出的，按权重排序。

### 5. 方言/地域特征
- 具体的方言词汇和表达习惯
- 地域性的价值观（如东北人的爽快、广东人的务实、上海人的精细）
- 饮食文化在语言中的体现

---

## 分析原则

1. **行为优先**：所有标签必须翻译为具体行为描述
2. **原材料为王**：如果原材料和标签推断矛盾，以原材料为准
3. **信息不足时**：标注 `[信息不足，根据{依据}推断]`，给出合理推断
4. **不美化不丑化**：真实还原，唠叨就是唠叨，严厉就是严厉
5. **年代感很重要**：一个50后和一个80后说同样意思的话，方式完全不同
