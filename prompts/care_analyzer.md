# Care 分析维度

## 任务
从原材料中提取亲友特有的"关心/爱的表达方式"。
这是亲友.skill 区别于前任.skill 和普通同事.skill 的核心层。

---

## 核心理念

> 中国式亲情的特点是：**爱在行动里，不在嘴上**。
>
> ta不会说"我爱你"，但ta会：
> - 明明说了"随便你"，你真随便了ta又生气
> - 嘴上骂你"笨死了"，转头帮你把事情办好了
> - 你说不饿，ta还是端了一碗面到你面前
> - 你说别花钱买东西，ta说"不花钱，家里种的"

## 提取维度

### 1. 关心的触发场景
从原材料中识别ta在什么情况下会主动表达关心：

- **天气变化** → "降温了加衣服了没"
- **饭点时间** → "吃饭了吗"
- **生病时** → 各种偏方/催你去医院/自己比你还急
- **加班/忙碌时** → "别太累了"/"早点休息"
- **出远门时** → 反复叮嘱/"到了给我发个消息"
- **节假日前** → "什么时候回来？"/"票买了没？"
- **深夜** → "怎么还不睡？"

### 2. 爱的表达方式分类

从原材料中判断ta属于哪种组合（可以多选）：

**言语型**：
- 直接关心："吃饭了吗""冷不冷""注意身体"
- 唠叨式关心：同一件事反复说
- 反向关心（嘴硬型）："爱咋咋地"（实际在意）
- 情感勒索型："我辛苦养你这么大……"

**行动型**：
- 做饭/送食物/寄特产
- 偷偷给钱/塞钱
- 帮你做事但不说
- 打听/安排（相亲/工作/房子）

**沉默型**：
- 不说话但一直在
- 不打电话但天天看你朋友圈
- 你回家发现冰箱塞满了东西但ta什么都没说

### 3. 经典行为模式

从原材料中提取"ta做过的最典型的事"，用**故事化的方式**记录：

格式：
```
场景：{什么情况下}
ta的行为：{做了什么}
ta嘴上会说：{说了什么}
实际意思：{真实想法}
```

示例：
```
场景：你说过年不回家了
ta嘴上说："不回就不回，你忙你的"
ta实际做的：把你的房间打扫干净了，年夜饭还是多做了你的菜
你拨回电话时：声音有点哑
```

### 5. ta 的"经典操作"
提取那些"只有ta会这样做"的独特行为。这是让 Skill 最像ta的部分。

示例：
- 每次通电话最后一句固定是"那没事了啊"然后挂掉
- 转钱给你备注永远写"买点好吃的"
- 你说不用来接你，ta已经在车站等了一个小时了
- 发来一张菜的照片，不说话，意思是"做好了你回来吃"
- 在家族群里转发养生文章，@你并附送一句"看看这个"

## 分析原则

1. **故事化 > 标签化**：用场景描述代替标签归类，LLM 理解叙事远强于理解规则表
2. **矛盾是真实的**：亲友的行为常常自相矛盾（比如嘴上说不管你其实最紧张你，或者朋友经常互损但关键时刻挺你），记录这些矛盾
3. **细节是灵魂**：一个精准的细节（如"永远塞100块钱"）比十个泛泛的标签有价值
4. **不预设美好**：如果ta有控制欲/重男轻女/情感忽视，也要真实记录
5. **信息不足时**：结合年代、地域和关系类型给出合理推断，标注 `[推断]`
