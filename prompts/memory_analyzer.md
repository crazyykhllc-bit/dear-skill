# Memory 分析维度

## 任务
从原材料中提取家庭记忆和共同经历，构建关系的事实性记忆，让对话有真实感。

---

## 提取维度

### 1. 家庭大事件时间线
从原材料中提取可以确定时间的重要事件：

- 搬家/盖房子
- 家人的生老病死
- 过年/团聚的时刻
- 你的人生节点（上学/工作/结婚）对ta的影响
- 家庭经济变化（下岗/做生意/退休）

### 2. 日常生活模式
- ta的作息习惯
- ta的日常活动（买菜/做饭/散步/打牌/看电视/种地）
- ta的特长（做菜/手工/种菜/修东西）
- ta的社交圈（邻居/老同事/牌友/广场舞伙伴）

### 3. 家庭饮食记忆
（在中国家庭中，食物是最重要的情感载体之一）

- ta的拿手菜
- 过年必做的菜
- ta给你做过的"你最想念的那口饭"
- 饮食习惯和讲究
- 特产/老家味道

### 4. 空间记忆
- 老家/老房子的样子
- 关键地点（ta工作的地方/常去的地方/家附近的标志物）
- 你回家的路线/场景

### 5. 节日与仪式
- 过年的传统（年夜饭/拜年/压岁钱）
- 清明/中秋/端午等节日的习惯
- 生日怎么过
- 家族特有的仪式或习惯

### 6. 家族故事
ta反复讲的那些故事：
- 自己年轻时的经历
- 家族的历史
- 关于你小时候的故事
- 人生教训故事

### 7. 共同记忆片段
你和ta之间最重要的记忆：
- 最开心的时刻
- 最难过的时刻
- 争吵/和解
- 印象最深的一次对话

### 8. 重要日期
- ta的生日
- 忌日（如果已故）
- 结婚纪念日
- 其他家庭重要日期

---

## 分析原则

1. 所有记忆必须基于原材料或用户口述，不得虚构
2. 时间信息尽量精确（从消息时间戳/照片 EXIF 推断）
3. 食物和地点记忆往往是最真实的情感锚点，优先提取
4. ta反复讲的故事特别重要——重复本身就说明这件事对ta很重要
5. 争吵/冲突的记忆同样重要——是真实关系的一部分
6. 如果信息不足，标注 `[待补充]` 而非推测
