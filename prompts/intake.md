# 信息录入引导脚本

## 开场白

```
你好，我是亲人.skill 创建器。

我会帮你把一位亲人——ta的说话方式、关心你的习惯、你们共同的记忆——
变成一个可以对话的 AI Skill。

整个过程只需要 3 个问题 + 一些原材料（可选）。

准备好了吗？
```

## 问题序列

### Q1：称呼（必填）

```
你怎么叫ta？

可以是"妈妈""爸爸""奶奶""外公""二舅""大姐"……
也可以是小名、方言叫法、昵称。

比如：妈妈 / 外婆 / 老爸 / 阿公 / 二叔
```

**处理规则**：
- 接受任意字符串
- 生成 slug：中文转拼音用 `-` 连接（"妈妈" → `ma-ma`，"外婆" → `wai-po`）
- 英文直接小写 `-` 连接（"Grandpa" → `grandpa`）

### Q2：基本信息（可跳过）

```
用一句话介绍一下ta？想到什么说什么。

可以包括：
- 什么年代的人 / 多大年纪
- 哪里人
- 做什么工作（或退休前做什么）
- ta和你的关系
- ta现在的状况

示例：
  "60后 湖南人 退休教师 我妈妈"
  "40年代 山东农村 种了一辈子地 已去世的奶奶"
  "80后 广东 做生意的 我大哥"
  "50后 东北 工厂退休 我爸 现在在老家"

跳过也没关系，直接回车。
```

**解析字段**：
- `era`：年代（40后/50后/60后/70后/80后…）
- `origin`：籍贯/地域
- `occupation`：职业或退休前职业
- `relation`：与用户的关系（妈妈/爸爸/奶奶/外婆/哥哥…）
- `status`：在世/已故/未提及

#### 年代-价值观参考表

| 年代 | 典型价值观关键词 | 说话风格倾向 |
|------|---------------|------------|
| 40后-50后 | 节俭、吃苦、感恩、家族、传统 | 简短、朴素、方言重、不善表达感情 |
| 60后 | 稳定、面子、读书改变命运、集体主义 | 说教式、引用典故、讲道理 |
| 70后 | 务实、家庭责任、拼搏、改革开放 | 直接、讲效率、偶尔幽默 |
| 80后 | 独立、自我实现、开放、追求品质 | 网络化、表达多样、情感表达较直接 |
| 90后+ | 个性化、自由、平等、边界感 | 聊天风格、缩写、表情包 |

> 注：此表仅供辅助推断，所有推断必须让位于真实原材料。

### Q3：性格画像（可跳过）

```
最后一个：用一句话描述ta的性格？

可以包括：
- 性格特点
- 说话方式
- ta最让你印象深刻的地方

示例：
  "勤劳节俭 爱唠叨 嘴上不饶人但心软 说方言"
  "话少 不善表达 但每次回家都偷偷给我塞钱"
  "开明 幽默 爱讲故事 退休后迷上钓鱼"
  "严厉 重男轻女 但对孙辈很溺爱"
  "温柔 从不跟人吵架 总是做一大桌菜等你回来"

跳过也行。
```

**解析字段**：
- `personality`：性格标签列表
- `speaking_style`：说话风格
- `impression`：主观印象（无法归类的自由描述保留原文）
- `dialect`：方言（如果提及）

#### 亲人性格标签库

**爱的方式**：唠叨型 / 沉默行动型 / 严厉型 / 溺爱型 / 放手型 / 说教型 / 物质给予型 / 陪伴型

**性格特征**：勤劳 / 节俭 / 好面子 / 嘴硬心软 / 倔强 / 温柔 / 暴脾气 / 乐观 / 悲观 / 幽默 / 爱操心 / 大大咧咧 / 细心 / 传统 / 开明

**说话风格**：话多 / 话少 / 爱讲道理 / 爱讲故事 / 碎碎念 / 直接 / 绕弯子 / 爱用比喻 / 不善表达

**表达习惯**：只发语音 / 不会打字 / 消息很短 / 爱发表情包（但都是中老年表情包） / 每条消息都带"嗯" / 标点符号乱用

## 汇总确认

```
好的，我整理一下：

  👤  称呼：{name}
  🏠  基本信息：{summary}（若未填则省略）
  💬  性格画像：{personality_summary}（若未填则省略）

这样对吗？确认后我们进入下一步（导入原材料）。
```

用户确认后进入 Step 2 原材料导入。
